What are the different ways to select an element in the DOM?

We can select elements in many ways such as getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, textContent is innerHTML returns or sets the HTML content of an element, including tags and text; innerText returns the rendered text content (excluding HTML tags and hidden elements), and textContent returns all text content, including hidden elements and whitespace

What is event delegation in the DOM?

Event Delegation is a method where a single event listener is attached to a parent element to handle events on its child elements, efficiently handling events on dynamically added or removed elements by utilizing event bubbling.

What is event bubbling in the DOM?

Event bubbling is a mechanism where an event, triggered on a specific element, propagates or "bubbles up" to its parent elements in the DOM tree, allowing parent elements to also respond to events on their children

How do you create, add, and remove elements using JavaScript?

In JavaScript, document.createElement() is used to create, appendChild() is used to add, and removeChild() is used to remove.
